About The Game

    This game contains some elements of RPG, the character can get EXP through defeating the monsters or completing the quests. And the new skills will unlock when the character level up. After completing the last quest and passing the trial map, the character will get powerful ability and be able to fight with the final boss.

How to play

  Player can use arrow keys to control the character's action.
